# 마크다운 문법

# 1.Heading

>  문서의 제목이나 소제목으로 사용



# 2. List

> 순서가 있는 리스트

1. First item
2. Second item
3. Third item
4. Fourth item
5. 

> 순서가 없는 리스트

- First
- Second
- Third
- Fourth

# 3.Fenced Code block

> 코드 블록에 특정 언어를 명시하면 Syntax Highlighting 적용 가능

```java
	public static void main(String[] args) {
System.out.println("안녕");
  }

```

# 4.Inline Code block

>특정 키워드 표기에 사용

- 키보드와 연결하는 클래스는 `Scanner`
- 해시태그 `#마크다운` `#코드블록`

# Link

> 문자열 (url)을 통해 링크를 작성 가능
