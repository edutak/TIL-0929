# Typora 문법 정리





### 볼드체와 기울임

**안녕하세요** 

*안녕하세요* 

__안녕하세요__

_안녕하세요_

#### How?

앞,뒤로 * 혹은 _ 를 2개 씩 사용하면 볼드체, 1개 씩 사용하면 기울임 표현



### List

#### 순서가 있는 리스트(ol)

1. 봄
2. 여름
3. 가을
4. 겨울

##### How

숫자와 마침표를 이용하여 ex) 1. 봄 (enter)

#### 순서가 없는 리스트(ul)

* 국어
* 영어
* 수학
* 과학

##### How

asterisks(*)를 이용하여 작성 ex) *국어 (enter)
